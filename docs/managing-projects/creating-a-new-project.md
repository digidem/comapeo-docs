## Starting a New Project


In the case where you want to share any observations from My Solo Project, but would rather collaborate with another device on a separate project, then Start a New Project.


> ### 👣 Step by Step  
>   
> _**Step 1:**_ Go to the <img src="/comapeo-docs/images/emojis/three-line-menu_351bd29be717f58f_78439734.png" alt="three-line-menu-black" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} />**Menu**.  
>   
> _**Step 2:**_ Tap on <img src="/comapeo-docs/images/emojis/participant_2dc5f8d1ce7cb4fc_78473365.png" alt="app-icon-participant" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} /> **Collaborate**.  
>   
> _**Step 3:**_ Select <img src="/comapeo-docs/images/emojis/coordinator_77ddbc4137798bed_78473392.png" alt="app-icon-coordinator" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} /> **Start New Project.**  
>   
> _**Step 4:**_ Add a **project name** and continue  
>   
> _**Step 5:**_ Select a **Project Statistics** option. This can be changed in Coordinator Tools  
>   
> _**Step 5:**_ Your new project is ready!


![project_stats_and_finish_setup.jpg](/images/creatinganewproject_1.jpg)


:::note 💡 Tip
The project that you started when first onboarding onto CoMapeo will still be available go back to
:::
Go to 🔗 [**Inviting Collaborators**](/docs/inviting-collaborators) for instructions
