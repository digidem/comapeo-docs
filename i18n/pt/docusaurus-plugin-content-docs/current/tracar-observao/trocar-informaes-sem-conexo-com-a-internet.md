# Usando o Exchange Offline


## Troca com Wi-Fi


---


A troca requer 4 coisas:

- Dois ou mais dispositivos com<img src="/images/emojis/comapeo_logo_ci_3a40065bd081188f_84653629.png" alt="comapeo_logo_circle" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} />CoMapeo,
- Os dispositivos estão conectados à mesma conexão WiFi

    :::note 🖼️
    static/images/usingexchangeoffline_0.jpg
    :::
- dispositivos pertencem ao mesmo projeto

    :::note 🖼️
    static/images/usingexchangeoffline_1.jpg
    :::
    <div class="notion-spacer" aria-hidden="true" role="presentation"></div>

- Novas observações ou trilhas
- Informações do projeto atualizadas

Vá para 🔗[**Entendendo Como a Troca Funciona**](https://notion.so/docs/understanding-how-exchange-works)para visão geral completa e detalhes


> ### 👣 Passo a Passo  
>   
> _**Passo 1:**_Abra o<img src="/images/emojis/three-line-menu_351bd29be717f58f_84653488.png" alt="three-line-menu-black" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} />Menu  
>   
> _**Passo 2:**_Toque<img src="/images/emojis/remote_archive_70c2034a3fd6001e_84653470.png" alt="app-icon-comapeo-remote-archiver" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} />_**Exchange**_  
>   
> _**Passo 3:**_A tela de troca exibirá "Dispositivos Encontrados" quando estiver corretamente conectada a um roteador.  
>   
> Dica: As configurações de troca podem ser ajustadas, se necessário.  
>   
> Vá para 🔗[Entendendo Como a Troca Funciona -> Configurações de Troca](https://notion.so/docs/understanding-how-exchange-works#adjusting-exchange-settings)  
>   
> ---  
>   
> _**Passo 4:**_Toque**Iniciar**para permitir que dispositivos conectados saibam que você está pronto para trocar dados do projeto  
>   
> _**Passo 5:**_Quando outros dispositivos se juntam ao Exchange, todas as novas observações e dados relevantes do projeto serão trocados.  
>   
> _**Passo 6:**_**Completo**será exibido quando todas as observações forem trocadas. Toque**Concluído**para retornar a<img src="/images/emojis/three-line-menu_351bd29be717f58f_84653488.png" alt="three-line-menu-black" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} />Menu


<div class="notion-spacer" aria-hidden="true" role="presentation"></div>


> 💡 **Dica:**A troca offline pode acontecer de forma bastante rápida se todos começarem a trocar aproximadamente ao mesmo tempo.

### Tendo Problemas?


Vá para 🔗[Solução de Problemas: Mapeamento com Colaboradores -> Problemas de Troca](https://notion.so/docs/troubleshooting-mapping-with-collaborators#exchange-problems)


Problemas comuns com a troca estão relacionados a conectar-se à mesma WiFi ao mesmo tempo, especialmente se o roteador ou ponto de acesso móvel não estiver conectado à internet. Muitas vezes, os dispositivos se desconectarão de uma fonte de WiFi para favorecer uma que tenha internet ou que esteja salva nas memórias do dispositivo. Estas são configurações que você pode verificar para reduzir problemas relacionados a conexões WiFi.
