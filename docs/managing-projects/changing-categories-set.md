## When is changing a Categories Set needed


After downloading customized categories sets


### Loading a new categories set file 


The file type **.comapeocat** is proprietary to CoMapeo. It cannot be opened by any other app not can be opened by Android. Use CoMapeo to load the category set file to a selected project. This ability is limited to those who are coordinators of projects.


:::note 💡
👣 > ### **Step by Step: Mobile**
>
> _**Step 1:**_ Look for the file in your downloads folder and transfer it to the Mobile phone you are using CoMapeo on. You can use an email, Google Drive, WhatsApp or the tool that works best for you to get files to your phone.
>
> ![2025-08-26_10-17.png](/images/changingcategoriesse_0.png)
>
> _**Step 2:**_ Open <img src="/comapeo-docs/images/emojis/comapeo_e67891bb2deee359_24651498.png" alt="comapeo-platform" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} /> CoMapeo and, in the main menu, **select the project** where the new category set in needed.
>
> _**Step 3:**_ _Open_ _**Coordinator Tools.**_
>
> > ⚠️ **Warning:** This is not visible to Participants of a project
>
> _**Step 3:**_ _Select_ **Project Categories**.
>
> ![1000000070.png](/images/changingcategoriesse_1.png)
>
> _**Step 4:**_ _Tap_ **Import Categories** to open the file picker of your operating system
>
> ![1000000071.png](/images/changingcategoriesse_2.png)
>
> _**Step 5:**_ Find the category set file (.comapeocat) and select it. CoMapeo will attempt import the file selected. If successful, information on the Catergories screen will be updated
>
>
>
> > 💡 Invite people in your team to your project, and they will have your categories once they join your project. If they are already on your project and you want them to have the updated Category Files, exchange information with them and this will update their categories.
> > Category sets are included in project data. Category sets are shared with with new teammates after invitation to projects are complete. They are are also included when doing Exchange with teammates.
>
> ---
:::
