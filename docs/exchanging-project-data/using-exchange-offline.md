### Exchange with Wi-Fi


---


Exchange requires 4 things:  

- 2 or more devices with CoMapeo,

    ![Screenshot_2025-09-02_at_9.53.15_AM.png](/images/usingexchangeoffline_0.png)

- devices are connected to the same WiFi connection

    ![pngtree-router-vector-icon-in-flat-style-png-image_6358749.jpg](/images/usingexchangeoffline_1.jpg)


    ![Screenshot_2025-09-02_at_9.53.33_AM.png](/images/usingexchangeoffline_2.png)

- devices belong to the same project
- new collected data

    ![Screenshot_2025-09-02_at_9.53.43_AM.png](/images/usingexchangeoffline_3.png)


 Go to 🔗 [**Understanding How Exchange Works**](/docs/understanding-how-exchange-works) for full overview and details


> ### 👣 Step by Step  
>   
> 1. Open the Menu  
>   
> 2. Tap _**Exchange**_  
>   
> 3. Exchange screen will display “Devices Found” when properly connected to a router. If needed, Exchange Settings can be adjusted. See below for instructions.  
>   
> 4. Tap “Start” to let connected devices know you are ready to exchange Observations.  
>   
> 5. When other devices join the Exchange, all new observations will be exchanged.  
>   
> 6. Complete” will display when all observations are exchanged. Tap “Done” to return to Menu


> 💡 **Tip:** Offline exchange can happen fairly quickly if everyone starts exchange around the same time.
