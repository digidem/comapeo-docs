## Exchange with Wi-Fi


---


Exchange requires 4 things:  

- Two or more devices with <img src="/images/emojis/comapeo_logo_ci_3a40065bd081188f_84653629.png" alt="comapeo_logo_circle" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} /> CoMapeo,
- Devices are connected to the same WiFi connection

    ![](/images/usingexchangeoffline_0.jpg)

- devices belong to the same project

    ![](/images/usingexchangeoffline_1.jpg)


    <div class="notion-spacer" aria-hidden="true" role="presentation"></div>

- New observations or tracks
- Updated project information

 Go to 🔗 [**Understanding How Exchange Works**](/docs/understanding-how-exchange-works) for full overview and details


> ### 👣 Step by Step  
>   
> _**Step 1:**_ Open the <img src="/images/emojis/three-line-menu_351bd29be717f58f_84653488.png" alt="three-line-menu-black" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} />Menu  
>   
> _**Step 2:**_ Tap <img src="/images/emojis/remote_archive_70c2034a3fd6001e_84653470.png" alt="app-icon-comapeo-remote-archiver" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} /> _**Exchange**_  
>   
> _**Step 3:**_ Exchange screen will display “Devices Found” when properly connected to a router.  
>   
> Tip: Exchange Settings can be adjusted if needed  
>   
> Go to 🔗 [Understanding How Exchange Works -> Exchange Settings](/docs/understanding-how-exchange-works#adjusting-exchange-settings)   
>   
> ---  
>   
> _**Step 4:**_ Tap **Start** to let connected devices know you are ready to exchange project data  
>   
> _**Step 5:**_ When other devices join the Exchange, all new observations and relevant project data will be exchanged.  
>   
> _**Step 6:**_ **Complete** will display when all observations are exchanged. Tap **Done** to return to <img src="/images/emojis/three-line-menu_351bd29be717f58f_84653488.png" alt="three-line-menu-black" className="emoji" style={{display: "inline", height: "1.2em", width: "auto", verticalAlign: "text-bottom", margin: "0 0.1em"}} />Menu


<div class="notion-spacer" aria-hidden="true" role="presentation"></div>


> 💡 **Tip:** Offline exchange can happen fairly quickly if everyone starts exchange around the same time.

### Having Problems?


Go to 🔗 [Troubleshooting: Mapping with Collaborators -> Exchange Problems](/docs/troubleshooting-mapping-with-collaborators#exchange-problems) 


Common issues with exchange relate to connecting to the same WiFi at the same time, especially if the router or mobile hotspot is not connected with the internet. Often devices will disconnect from a WiFi source to favor one that has internet,  or is saved in the device memories. These are setting you can check to reduce issues related to WiFi connections.
